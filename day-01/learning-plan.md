## What DevOps Means to Me

DevOps is the bridge between writing code and actually running it reliably in production.

I've always handed off deployments to "someone else."
DevOps is me finally taking ownership of that entire lifecycle.

Cloud Engineering = building and managing infrastructure that scales, without needing physical servers.

---

## Why I'm Starting This

- Been writing code for a while, but I only own the application layer
- I want to own the **full stack** - from code to production
- This is a **deliberate career move**, not a side hobby
- DevOps & Cloud roles = interesting problems + real career growth (hard to ignore 😅)
